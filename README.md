## Getting started

```sh
# 1. Clone this repo
git clone https://github.com/kelseyyim/CrossyRoadAlpha.git

# 2. Open up main project file with Unreal Engine Editor
CrossyRoadAlpha.uproject

# 3 Press Play button and right click on the game screen

```
